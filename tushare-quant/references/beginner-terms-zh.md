# Beginner Terms

Use this reference when explaining A-share quant reports to a beginner.

| Term | Plain Chinese explanation |
| --- | --- |
| MA / moving average | 均线。把最近若干天价格取平均，用来减少日内波动噪音。 |
| MACD | 趋势和动能指标。DIF 高于 DEA 常被叫做多头信号，但它滞后，不等于一定上涨。 |
| KDJ | 更敏感的短线指标。J 值很高可能短线过热，很低可能短线超卖。 |
| Return | 收益率。投入资金最后赚亏的百分比。 |
| Max drawdown | 最大回撤。从历史高点跌到后续低点的最大跌幅，用来衡量中途最难受的亏损。 |
| Win rate | 胜率。赚钱交易占全部交易的比例。高胜率不一定赚钱，因为单次亏损可能更大。 |
| PE | 市盈率。市场愿意为公司每 1 元利润付多少钱。 |
| PB | 市净率。股价相对每股净资产的倍数。 |
| ROE | 净资产收益率。公司用股东的钱赚钱的效率。 |
| Volume | 成交量。当天成交了多少股或手，反映交易活跃程度。 |
| Turnover | 换手率。流通股里有多少比例在当天换了人。 |
| Slippage | 滑点。真实成交价比回测假设价格更差的部分。 |
| Survivorship bias | 幸存者偏差。只看现在还活着的股票，会高估过去策略表现。 |
| Future leakage | 未来函数。回测时用了当时不可能知道的数据，会让结果虚高。 |
